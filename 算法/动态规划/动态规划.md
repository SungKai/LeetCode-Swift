# 动态规划

Dynamic Programming（DP）

动态规划中**每一个状态一定是由上一个状态推导出来**的，**这一点就区分于贪心**，贪心没有状态推导，而是从**局部直接选最优**

就比如在学习贪心算法的时候接触到的，往一个背包里面装东西，不是每次在一堆东西里面选到体积最大的，就能使价值总和最大的。

## 解题步骤

**状态转移公式（递推公式）很重要**

1. 确定dp数组（dp table）以及下标的含义
2. 确定**递推公式**
3. dp数组如何**初始化**
4. 确定**遍历顺序**
5. **举例**推导dp数组

## 如何debug

**把dp数组打印出来，看看究竟是不是按照自己思路推导**
